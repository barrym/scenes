h1. Scenes

Scenes is a Rails plugin to set up system scenes, mainly for use with "Cucumber":https://github.com/aslakhellesoy/cucumber/tree, but can also be used to set your system into a preset state.

h2. Usage

h3. Characters

<pre>
<code>
  Scenes::Character.named("Alice) {
    User.create(:name => 'Alice')
  }
</code>
</pre>

h3. Scenes

<pre>
<code>
  Scenes::Scene.named("Alice and Bob") {
    Scenes::Character['Alice']
    Scenes::Character['Bob']
  }
</code>
</pre>
